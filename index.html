Clean the kubernetes YAML and chart mess

A carefully crafted language and developer tools for building fast.

.

clean8s is based around
the kcl language and kcl Studio, the Web IDE/debugger.

For beginners

  • A web IDE that lets you mount and inspect inside containers and easily write code
  • One-click container image building & registry setup
  • A library that covers casual deployment workflows without configuration

For DevOps professionals

  • Debug internal Kubernetes API calls
  • Create custom Controllers and dynamic resources
  • Extend and transform deployment plans in any programming language
Shaped by case studies of Terraform and Pulumi, and the UX of drag-n-drop deployments of the early Web.

The syntax and beautiful diagnostics of Terraform/HCL

kcl uses the same parser as languages in the Hashicorp family and embraces its block-based static strongly typed nature.
It also introduces namespacing syntax and a refined stdlib unlike Terraform it offers a less assertive execution strategy that gives the user more abilities to influence deployments.

The flexibility during execution of Pulumi

kcl offers a simple guided CLI and lets the user pick a Turing-complete language for maximum flexibility: you can dig under the abstractions and rewrite kubernetes API queries and plans as you wish.
